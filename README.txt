===========================
 8192 Game with High Score
===========================

Description:
------------
This is a simple browser-based version of the popular 2048 game, built using HTML, CSS, and JavaScript. The game now features a maximum tile of 8192, along with animations for tile movement, a restart button, and a local storage high score tracking feature.

How to Play:
------------
1. Use the arrow keys (Up, Down, Left, Right) to slide the tiles.
2. Combine tiles with the same numbers to add them together.
3. Reach the tile with the value 8192 to win the game!
4. If the board fills up and no more moves are possible, you lose.

Features:
---------
- Smooth tile movement animations.
- A 'Game Over' screen when no more moves are possible.
- A 'You Win!' alert when you reach the 8192 tile.
- Local storage-based high score tracking, which saves the highest score across sessions.
- Clean and responsive design.

Files:
------
1. **index.html** - The main HTML structure of the game.
2. **styles.css** - Contains all the styles for the game, including the grid layout and tile colors.
3. **script.js** - The main game logic, handling tile movements, scoring, high score storage, and game over detection.

Installation:
-------------
1. Download all the files: `index.html`, `styles.css`, and `script.js`.
2. Place them in the same folder.
3. Open the `index.html` file in any modern web browser to start playing.

Requirements:
-------------
- A modern web browser (Google Chrome, Firefox, Safari, Edge, etc.)
- No additional libraries or frameworks are required.

High Score:
-----------
- The game automatically tracks the highest score achieved using local storage.
- The high score is displayed on the game interface and is retained across browser sessions.

Restarting the Game:
--------------------
- To restart the game, click the "Restart" button or the restart button shown on the Game Over screen.
